# FreshExpress 项目 README

## 项目概述

FreshExpress 是一个现代化的生鲜配送管理系统，采用前后端分离架构设计，提供完整的员工管理、分类管理、菜品管理、套餐管理等功能模块。

## 技术栈

### 后端技术
- **Spring Boot**：核心框架
- **MyBatis**：持久层框架
- **MySQL**：数据库存储
- **Redis**：缓存服务
- **JWT**：身份认证
- **PageHelper**：分页插件
- **AspectJ**：面向切面编程
- **Lombok**：简化代码

### 前端技术
- **Vue.js**：前端框架
- **Element UI**：UI组件库
- **Axios**：HTTP客户端

## 项目结构

```
fresh-express-server/
├── fresh-common/     # 公共模块
├── fresh-dto/        # 数据传输对象
├── fresh-mapper/     # 数据访问层
├── fresh-pojo/       # 实体类
└── fresh-server/     # 服务启动模块
```


## 核心功能

### 员工管理
- 员工登录认证
- 员工信息管理
- 权限控制

### 分类管理
- 菜品分类管理
- 套餐分类管理
- 分类状态管理

### 菜品管理
- 菜品信息维护
- 菜品分类关联
- 菜品状态管理
- 菜品口味管理

### 套餐管理
- 套餐信息维护
- 套餐菜品关联
- 套餐状态管理

## 系统特性

- **安全性**：采用 MD5 加密、JWT 认证
- **可扩展性**：模块化设计，易于扩展
- **高性能**：使用 Redis 缓存，提升系统性能
- **易维护**：代码结构清晰，注释完整
- **公共字段自动填充**：通过 AOP 切面自动填充创建时间和更新时间等字段
- **统一异常处理**：全局异常处理机制
- **数据校验**：全面的数据验证机制

## 快速开始

### 环境要求
- JDK 8+
- MySQL 5.7+
- Redis
- Node.js (前端)

### 部署步骤

1. **数据库配置**
   - 创建数据库
   - 执行数据库脚本

2. **后端部署**
   - 修改 `application.yml` 中的数据库连接信息
   - 启动 Spring Boot 应用

3. **前端部署**
   - 安装依赖：`npm install`
   - 启动项目：`npm run serve`

## 项目亮点

- **统一异常处理**：完善的异常处理机制
- **分页查询**：支持条件分页查询
